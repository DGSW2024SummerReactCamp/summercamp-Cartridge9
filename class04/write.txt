배운 내용 정리
1. 스프레드 연산자 ( 전개 구문 ) - ( ...배열/문자열 ) : 반복 가능한 개체를 활용하게 해주는 것
  concat(), splice() 등을 대체해서도 사용 가능 : [...parts1, ...parts2]
1-2. 얕은 복사와 깊은 복사 : 얕은 복사는 복사 전 값을 참조하여 복사되는 것 / 즉, 메모리에서 같은 공간을 가르킴 / 하나를 변경하면 다른 하나도 변경될 수 있음
                            깊은 복사는 원본과 복사본이 아예 별개로 작동하는 복사 / 복사된 객체들은 구조적으로 같음
2. useState : 컴포넌트에 state 변수 추가 / state 값과, state 값을 업데이트해주는 set 함수 반환 / set 함수는 다음 렌더링에 업데이트됨
3. useReducer : state의 상태를 넘겨주면 해당 상태에 맞는 값으로 변화시킬 수 있는 훅 / dispatch 함수를 통해 state를 업데이트시킴
  state 값을 받아온 뒤, 객체를 받아와서 값 변경시키거나 하는 등의 활동 수행 가능
4. useRef : ref 값을 변경해도 컴포넌트는 리렌더링되지 않음 / 즉 시각적으로 보이는 것이 아닌 정보 저장에 용이함 / 컴포넌트에 로컬로 저장됨
